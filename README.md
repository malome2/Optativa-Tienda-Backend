# Tienda Optativa

- Descripcion
- Stack: NodeJs, MongoDB, Node, Git, Docker, Docker-Compose, Postman, Docker compas

2. Usuario no logueado
3. Administrador

- añadir productos al carrito
- quitar productos del carrito
- Log in 
## Req. Funcionales Usuario Logueado
- ver productos
- busca productos
- añadir productos al carrito
- quitar productos del carrito
- Comprar carrito
